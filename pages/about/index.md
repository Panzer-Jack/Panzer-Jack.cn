# 关于我

我目前是一名来自中国上海的刚毕业没多久的大学生（划掉，已经毕业了），主修计算机网络. 平时喜欢做一些有趣的开源项目并且学习一些好玩的东西。比如一些能在生活中为我节省寿命的物联网小工具，奇奇怪怪的小网站 ~(比如赛博女友)~ 等。

我喜欢很多东西 ~（是个高贵的二次元御宅）~ 比如看番、轻小说、Galgame, 等一切ACGN领域的元素。目前也在自学电子绘画和音乐。

未来准备做一个偏前端的 Web Dev，来实现经济自由（ 做更多我感兴趣的事情 ）

我是一个现实主义的梦想家，曾经的我有许多梦想：比如开一个兴趣工作室做一些有趣的东西，比如成为一名独立游戏制作人，或者像命运之之门的“胸针”一样找到一群志同道合的Lab member做一些有趣的小发明。确实，在大学前两年，我非常幸运遇到了如今我的Lab member 和2个人非常好的老师，给了我们一个小实验室来给我们当“避难所”。 前两年里我也非常愉快的过着梦想中的中二般的 mad scientist 生活，（如今想想穿着白大褂在校园里走来走去，突然感觉有点羞耻。。。。(＊／ω＼＊) ）但是如今也马上进入大学的后半段，终究也是得面对如今所有计算机专业的学生都得面对的现实，我并不是有着像胸针一样男主角光环，没有发现 Time Machine 微波炉 的运气，没有像无敌hacker 桶子一样的超强技术, 更没有一个天才助手的女主角（悲）。有的只是能看到Lab member各奔东西的不远的未来。毕竟我们这几个都不是同一个专业的，有机械、电子、嵌入式、计算机等等，大家因为兴趣和竞赛凑合在一起，也正会因为这些走向不同的路，因此不免也得朝前看，仔细思考自己的未来。

那么放弃吗？自己的理想主义。肯定不啊，ο(=• ω <=)づ☆.。，即使如今可能需要暂时搁置掉这些，但在不远的未来，只要有可能，我依然会去寻找属于我自己的理想 Life Style。

到最后的最后我想附上一段 おけまる 老师 在 《夢見る男子は現実主義者》的第六卷后记里写的一段话:

---

( 摘自 《夢見る男子は現実主義者》的第六卷后记 )

阅读本作品的人当中，我认为应该有十来岁与尚未就业的人。其中想必也有人悄悄地将成为『轻小说作家』列入未来的梦想候选名单吧。偷偷告诉大家，我在找工作前就是抱持这样的想法。当时的我自认「自己从高中开始就在写作，说不定能凭借这项优势，从事与写作相关的工作」。

然而，从现实的角度来看，就会发现事情并没有那么单纯。在这个找工作大不易的时代，把进入门槛更高的轻小说作家当成第一选择，在精神上是一件很难受的事。我对自己的将来担心得不得了。原本就有「该如何向家人开口……」这样的课题，我没有勇气抬头挺胸追寻自己的目标。

最终──求职结束后，我进入一家普通企业就职。更一度放弃了轻小说作家的道路。

不过，即便如此，我的内心依然怀有继续写作的欲望。因为我在高中原本就有写作的习惯。就算不能当饭吃，我也想一直坚持这个兴趣，光是在网站上让读者感到「有趣」，就能让我每天充满写作的动力。

在这样基于兴趣持续写作的过程中，我经常投稿作品的网站突然提出举办比赛的企划。那就是成为本作品问世契机的『HJ网路小说大赏2019』。每个投稿小说的网站，每年都会实施几次像这类与文库合作的企划。例如HJ文库、HJ文库，还有HJ文库等。

那次的比赛获奖，成为我完成轻小说作家梦想的一大契机。我记得当时有一段时间还没有真实感，完全没想到自己能在踏入社会的这个时候实现当初的梦想。

经过前面的说明，我想大家应该都明白了。没有必要急着成为轻小说作家，成为轻小说作家没有年龄上的限制。找到工作不代表自己今后的人生就这么被定型。要是学生时代的我明白这个道理，或许就能以更轻松的心情踏入社会也说不定。当初如果有人告诉我的话，我就不用兜那么一大圈了。

我认为脚踏实地生活是非常重要的一件事。可是，就这么持续下去，大家会感到「满足」吗？我想，应该没人希望到了一定年纪才感叹自己什么梦想都没能实现。如果读者中有人对成为轻小说作家怀抱着憧憬，请千万不要放弃，试着挑战自己的梦想。也可以像我一样先找个公司就业，有稳定的工作之后再说。梦想不会破灭，眼前的道路要挑战几次都可以。听到这里，应该给各位注入了一剂强心针吧。前方就是兼具梦想和现实的世界。满怀美梦的现实主义者这句话打从一开始就不存在矛盾。

期待有一天我能以读者的身分参与各位的作品。

以上，我是おけまる。
